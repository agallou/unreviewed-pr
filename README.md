Unreview PR
===========

Displays a list of unreview PR in an HipChat Connect Glance.

Install
-------

* `docker-compose build`
* `docker-compose up`
* `./docker/bin/launch-bash`

    * `composer install`
    * `./vendir/bin/phinx migrate`

Dev tools
---------

* Connect to psql : `/docker/bin/psql`
